Quick Setup Guide

01

Download the Executable

Head over to the GitHub Releases page and grab the latest CeoShoot.exe standalone binary. No heavy installers required.

02

Run the Initial Setup

Launch the app. Our frameless Setup Wizard will let you configure Windows autostart. Settings are instantly saved to your Documents folder.

03

Shoot the Screen

Press PrintScreen anywhere. Drag to select your area, annotate if needed, and hit Ctrl + C to copy or Ctrl + S to save.
